# Description

This Ansible playbook that provides a very basic setup for all things Ubuntu server. This will make sure that there are some sane defautls, and also ensure that packages are installed and updated.

## Requirements:

- Ansible 2.x
- Test-Kitchen
- Inspec

- ubuntu-18
